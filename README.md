Getting started
===============

1. create a venv

`python -m venv venv`

2. activate venv

`source venv/bin/activate`

3. install requirements

`python -m pip install -r requirements.txt`

4. run tests

`python -m unittest -v`

5. run the app

`python -m starter_app`
